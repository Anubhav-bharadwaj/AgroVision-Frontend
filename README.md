# AgroVision

---

## 🚀 Overview

AgroVision is a frontend-only prototype of a plant disease detection web application built during a hackathon under the Web Development domain.

The goal was to design a clean, user-friendly interface that simulates how an AI-powered plant disease detection system would work — without implementing the actual machine learning backend due to time constraints.

---

## ⚠️ Important Note

This project currently includes only the frontend (UI/UX).  
The disease detection results are simulated (mock data) and not generated by a real AI/ML model.

---

## ✨ Features

- 🌱 Modern, responsive UI design  
- 📸 Drag & drop image upload interface  
- 🔍 Simulated disease analysis workflow  
- 📊 Confidence score visualization  
- 🧪 Multiple disease predictions display  
- 💊 Treatment suggestions:  
  - Chemical  
  - Organic  
  - Preventive measures  
- 📄 Export report (print feature)  
- 📱 Mobile-friendly layout  

---

## 🧠 How It Works (Frontend Logic)

1. User uploads a leaf image  
2. Image preview is displayed  
3. On clicking Analyze, a loading animation appears  
4. A random disease result is selected from a mock dataset  
5. UI dynamically displays:  
   - Disease name  
   - Confidence level  
   - Severity  
   - Treatment suggestions

- Confidence score
